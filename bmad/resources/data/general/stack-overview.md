# Stack Overview

## Backend
- FastAPI (Python 3.11)
- Supabase (Postgres, Auth, Storage)
- Redis (caching, pub/sub)
- pgvector (vector search)

## Frontend
- Next.js (React, TypeScript)
- Tailwind CSS
- Storybook

## Integraties
- Langchain (AI/LLM)
- Sentry (monitoring)
- Prometheus/Grafana (metrics)

## CI/CD
- GitHub Actions
- Docker Compose
